## Start project

- create file .env
- Add ENVs
  - ACCESS_TOKEN_SECRET
  - DATABASE_URL="mongodb+srv://<email>:<password>@cluster0.o0dhq6m.mongodb.net/professorChatGPT"
  - NEXT_PUBLIC_ORGANIZATION
  - NEXT_PUBLIC_API_KEY_GPT
- npx prisma generate
- npx prisma db seed
- npm run dev
- locale: http://localhost:3000
